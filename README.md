# 🚀 Portfolio - Sthevan Santos

Portfólio pessoal unificado com projetos, currículos e informações de contato.

## 📋 Conteúdo

- **Portfólio** — Site profissional com projetos, skills e formulário de contato
- **Currículos** — Versões em HTML/PDF por área de atuação (Desenvolvimento Web e Elétrica/Automação)

## 🛠️ Tecnologias

- **Next.js** — Framework React
- **TypeScript** — Tipagem estática
- **Tailwind CSS** — Estilização
- **Framer Motion** — Animações

## 🚀 Instalação

```bash
pnpm install
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## 📁 Estrutura

```
├── src/                    # Código Next.js
│   ├── app/               # Rotas e páginas
│   ├── components/        # Componentes React
│   └── lib/               # Utilitários e i18n
├── public/
│   ├── curriculo/         # Currículos (HTML + PDF)
│   │   ├── index.html     # Seletor de currículos
│   │   ├── programacao/   # CV Desenvolvimento Web
│   │   └── eletrica/      # CV Elétrica / Automação
│   └── ...
└── ...
```

## 📄 Currículos

Os currículos estão em `/curriculo`:
- 💻 [Desenvolvimento Web](public/curriculo/programacao/)
- ⚡ [Elétrica / Automação](public/curriculo/eletrica/)
